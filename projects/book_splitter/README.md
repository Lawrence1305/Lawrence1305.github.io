# 📚 书籍页面分割工具

纯前端静态网站，无需后端服务器，所有处理在浏览器本地完成。

## 功能特性
- ✅ 纯静态HTML，可直接打开使用
- ✅ 智能检测页面边界
- ✅ 可调节检测灵敏度
- ✅ 实时预览分割结果
- ✅ 生成标准PDF文档
- ✅ 隐私安全，数据不上传

## 使用方法
1. 直接用浏览器打开 `index.html`
2. 上传包含多页的书籍图片
3. 调整检测灵敏度(可选)
4. 点击"开始处理"
5. 预览结果后下载PDF

## 部署方式
### 方式1: 本地使用
直接双击 `index.html` 打开

### 方式2: GitHub Pages
1. 上传到 GitHub 仓库
2. 在仓库设置中启用 GitHub Pages
3. 访问: `https://你的用户名.github.io/仓库名`

### 方式3: Netlify/Vercel
拖拽整个文件夹到平台即可部署

## 浏览器要求
- Chrome 90+
- Firefox 88+
- Safari 14+
- Edge 90+

## 技术栈
- 原生 JavaScript (ES6+)
