## 网站结构

- `index.html` - 网站首页
- `about.html` - 关于我的页面
- `projects.html` - 项目展示页面
- `blog.html` - 博客文章列表
- `contact.html` - 联系方式页面
- `css/` - 样式文件目录
- `js/` - JavaScript文件目录
- `images/` - 图片资源目录
- `blog/` - 博客文章目录
